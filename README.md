# rest-api-thirdparty
Example REST API third party.

### REST api
List of routes:

| Route                 | HTTP    | Descriptions                                    |
| :-------------------- | :------ | :---------------------------------------------- |
| `/upload`             | POST    | Post file from external. file => folder upload  |

---
### Usage
```
- install dependencies => npm install
- run => npm start
- run with nodemon => npm run dev

```

Access the website http://localhost:2020
